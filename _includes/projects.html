Open Source Contributions

  • AWS Cloud Development Kit (CDK) - Framework for defining cloud infrastructure in programming languages such as Typescript. Contributor to various AWS service modules inside the CDK.

  • Backstage - Framework for building internal developer portals. Contributed a library that standardizes how Backstage plugins authenticate to AWS APIs. Previously maintainer of the AWS Proton plugin for Backstage.

  • Spinnaker - Multi-cloud continuous delivery solution. Contributor to the Amazon ECS plugin inside Spinnaker. Previously co-lead for the Spinnaker AWS SIG.

  • Flux - Continuous delivery solution for Kubernetes. Published a Kubernetes controller to enable Flux support for AWS CloudFormation.

  • GitHub Actions - Continuous delivery solution from GitHub. Published four GitHub Actions that form a workflow for continuously deploying to Amazon ECS.

  • Run MCP servers with AWS Lambda - Library for running legacy stdio-based MCP servers in Lambda.

  • img2lambda - Tool for repackaging Docker images into AWS Lambda layers.

  • oam-ecs - Tool for deploying Open Application Model configuration to Amazon ECS.

Demos & Sample Code

  • Amazon Bedrock Serverless Prompt Chaining - Generative AI application that demonstrates prompt chaining, orchestrated by AWS Step Functions.

  • AWS re:Invent Trivia Game (Live Demo) - Multi-tier application for answering re:Invent trivia questions, with static site, backend API service, and Slack chat bot.

  • clare-bot - GitHub pull request bot that provisions continuous integration preview environments.

  • Summarize My Document - Generative AI application that summarizes documents using a large language model and generates word cloud images.

  • Simple Calculator Service - Calculator application to demonstrate continuous integration and continuous delivery.

  • Continuous Integration Rick - Reports a Rick-and-Morty-themed build status to a GitHub pull request.

  • Grace Hopper Jeopardy - Jeopardy-style game with Grace Hopper-related trivia questions.

  • /gimmefoodtrucks - Slack command that returns that current day's local food trucks in Seattle.

  • /gimmepizza - Slack command that orders a pizza from Domino's.
